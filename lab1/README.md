# Упрощённая модель данных для учебного проекта: «Инвентаризация склада»

## Ключевые сущности

1. **Товар** — основной объект учёта, каждый товар имеет уникальный UUID идентификатор, название и цену.
2. **Категория** — группирует товары по видам (например, «Продукты», «Электроника»).
3. **Поставщик** — источник поставки товаров, указывается при приходе.
4. **Склад** — место хранения товаров, используется для определения фактических остатков.
5. **Приход** — документ, фиксирующий поступление товаров от поставщика.
6. **Расход** — документ, отражающий отпуск товаров со склада (продажа или списание).
7. **Остатки** — таблица текущего количества товаров на складе.

## Атрибуты сущностей

| Сущность           | Основные атрибуты                                          |
| ------------------ | ---------------------------------------------------------- |
| **Товар**          | id (UUID), name, price, category_id (UUID)                 |
| **Категория**      | id (UUID), name                                            |
| **Поставщик**      | id (UUID), name, contact                                   |
| **Склад**          | id (UUID), name, address                                   |
| **Приход**         | id (UUID), supplier_id (UUID), warehouse_id (UUID), date   |
| **Приход_позиция** | id (UUID), incoming_id (UUID), product_id (UUID), quantity |
| **Расход**         | id (UUID), warehouse_id (UUID), date                       |
| **Расход_позиция** | id (UUID), outgoing_id (UUID), product_id (UUID), quantity |
| **Остатки**        | product_id (UUID), warehouse_id (UUID), quantity           |

## Связи между сущностями

- Категория 1:N Товары
- Поставщик 1:N Приходы
- Склад 1:N Приходы и 1:N Расходы
- Приход 1:N Позиции прихода
- Расход 1:N Позиции расхода
- Товар 1:N Позиции прихода, 1:N Позиции расхода
- Товар 1:N Остатки, Склад 1:N Остатки
